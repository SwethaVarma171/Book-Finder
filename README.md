
# 📚 Book Finder

A React-based application that helps users find books using the Open Library API.

---

## 🛠️ Tech Stack / Languages Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **React.js (with Vite)**
- **Open Library API**

---

## 🚀 Features
- Search for books by title
- View book cover images
- Responsive design
- Fast search using API
